And here are the specs:

The Central Processing Unit (CPU) is an AMD Ryzen 5 5600G.

The Graphics Processing Unit (GPU) is an AMD Radeon 6650XT.

The motherboard is a Gigabyte Aorus Elite WiFi with the AM4 socket and an X570 chipset.

The Random Access Memory (RAM) is two 8 gigabyte sticks of DDR4 running at 3200 mega transfers per second.

The Power Supply Unit (PSU) is an NZXT C650 650-watt 80+ Bronze.
